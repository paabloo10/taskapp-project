# TaskFlow

Aplicación web de gestión de tareas desarrollada con HTML, Tailwind CSS y JavaScript.

## Funcionalidades

- Añadir tareas
- Eliminar tareas
- Guardado automático en LocalStorage
- Persistencia de datos al recargar la página
- Buscador de tareas

## Tecnologías

- HTML5
- Tailwind CSS
- JavaScript
- LocalStorage
